# 📔 NoteVault – The Ultimate Study Notes Library

**NoteVault** is a premium, enterprise-grade cloud platform designed for modern educational institutes and students. It provides a seamless, aesthetically pleasing interface to manage, read, and share study materials.

![NoteVault Banner](https://images.unsplash.com/photo-1544377193-33dcf4d68fb5?auto=format&fit=crop&q=80&w=1200)

## ✨ Premium Features

### 👨‍🎓 For Students
- **Smart PDF Viewer:** High-performance PDF rendering with zoom and page controls.
- **Focus Timer:** Built-in Pomodoro timer (25:00) to keep you productive.
- **Rich Text Editor:** Write your own notes with a professional formatting suite (Quill.js).
- **Audio Synthesis:** Listen to your notes using built-in Text-to-Speech (TTS).
- **Bookmarks:** Save your favorite materials for quick access.
- **Quick Share:** Instantly share notes via WhatsApp.

### 🏫 For Institutes (Enterprise)
- **Cloud Storage:** Securely upload and store PDFs using Firebase & Cloudinary.
- **White-Label Ready:** Customizable branding and institute names.
- **Admin Dashboard:** Full control over users and shared materials.
- **Monetization:** Built-in pricing plans and secure checkout simulation.

## 🛠 Tech Stack

- **Frontend:** HTML5, Vanilla CSS3 (Glassmorphism), JavaScript (ES6+)
- **Backend/Auth:** Google Firebase (Firestore, Auth)
- **Media Storage:** Cloudinary API
- **PDF Engine:** PDF.js
- **Rich Text:** Quill.js
- **Export:** Html2Pdf.js

## 🚀 Deployment

NoteVault is optimized for zero-config deployment on **Netlify**. 

1. Push code to GitHub.
2. Connect Repo to Netlify.
3. Your site is live!
