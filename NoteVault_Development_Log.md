# 📔 NoteVault Development Log & Project Summary

This document serves as a complete record of the development of **NoteVault**, a premium Study Notes Library platform, developed for **Rizwan Khan**.

---

## 🚀 Project Overview
- **Project Name:** NoteVault
- **Live URL:** [https://notevault-rizwan.netlify.app](https://notevault-rizwan.netlify.app)
- **GitHub Repository:** [Rizwan-Khan-2002/NoteVault](https://github.com/Rizwan-Khan-2002/NoteVault)

---

## 🛠 Development Journey & Milestones

### 1. Foundation & Core Features
- Initialized a single-file SPA architecture for maximum speed and simplicity.
- Integrated **Firebase Authentication** (Login/Signup) and **Firestore** for real-time data.
- Built a **Rich Text Editor (Quill.js)** and **Cloud PDF Upload** system.

### 2. Aesthetic & Premium UI Refinement
- **Glassmorphism:** Implemented a modern dark-themed UI with subtle glows and gradients.
- **3-Column Navbar:** Optimized the desktop view for professional navigation.
- **Dynamic Themes:** Added Light/Dark mode toggles.

### 3. Student & Institute Features
- **Focus Timer:** Built a Pomodoro timer (25:00) accessible from the navbar.
- **TTS (Text-to-Speech):** Integrated "Listen to Note" functionality.
- **Pricing Plans:** Simulated an enterprise pricing model for institutes (Starter, Pro, Enterprise).
- **Recent Activities:** Added a "Continue Reading" section on the home page.

### 4. Mobile Responsiveness (The iPhone 12 Pro Max Optimization)
- **Sidebar Drawer:** Replaced the bottom nav with a premium slide-in right sidebar for better reach and clarity.
- **Hamburger Menu:** Simplified the top navbar for small screens.
- **Floating Action Button (FAB):** Added a quick-action button for writing notes and uploading PDFs on mobile.
- **Timer Visibility:** Repositioned the focus timer to ensure it is always visible and doesn't overlap with other elements.
